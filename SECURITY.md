# Security Policy

## Supported Versions

We provide security updates for the latest released version only. Older versions are not supported.

## Reporting a Vulnerability

Please report security issues by email:

- Email: security@example.com

Include:
- A detailed description of the issue.
- Steps to reproduce.
- Any proof-of-concept code or screenshots.
- The affected version(s).

## Response Expectations

We will acknowledge receipt and provide an initial assessment timeframe. We do not guarantee specific remediation timelines, but we will share progress updates as the investigation proceeds.

## Coordinated Disclosure

Please do not disclose vulnerabilities publicly until we have confirmed a fix or agreed on a coordinated disclosure timeline.
